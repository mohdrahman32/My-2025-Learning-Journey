Deep Learning

Road Sign Classification (CNN)

Deep learning project using a custom CNN to classify road signs for autonomous and ADAS-related applications.

Road Sign Classification using a Custom CNN

Deep Learning & Computer Vision Project | Oct 2025

Overview

Designed and deployed a complete deep learning pipeline to automatically classify 30 different road sign classes, a critical task for ADAS and autonomous driving systems. The project covers model design, training optimization, and real-time inference via a web application.

Tools & Technologies
Python TensorFlow Keras CNN Streamlit Matplotlib
Methodology & Model Design
  • Designed a custom Sequential CNN architecture from scratch
  • Applied image preprocessing and normalization
  • Used data augmentation (rotation, zoom, shifts) to improve generalization
  • Implemented Dropout layers to reduce overfitting
  • Optimized training using Adam optimizer and Categorical Cross-Entropy loss
  • Applied Early Stopping (training stopped at Epoch 34)
Model Performance & Challenges
  • Successfully classified 30 traffic sign categories
  • Achieved strong validation performance on unseen test images
  • Faced challenges distinguishing visually similar classes (e.g., speed limit signs), highlighting real-world CV complexity
Deployment & Application
  • Built an interactive Streamlit web application
  • Users can upload an image and receive instant predictions
  • Demonstrates end-to-end ML deployment, not just model training
Key Learnings & Outcomes
  • Gained hands-on experience building CNNs from scratch
  • Understood the impact of data augmentation and regularization
  • Learned practical limitations of custom models in production scenarios
  • Identified transfer learning as a key improvement path
